CODE OF HYPERLINKS

Summary

1. a标签定义超链接，用于从一张页面链接到另一张页面。
2. a元素最重要的属性是href属性，它指示链接的目标。
3. 被链接页面通常显示在当前浏览器窗口中，除非您规定了另一个目标（target 属性）。
4. 可以使用 CSS 来设置链接的样式。
5. name属性
6.绝对连接，相对连接，图片链接。
